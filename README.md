# odin-recipes

## Description
This is a basic recipe website built as part of The Odin Project's Foundations curriculum. The primary objective of this project is to practice and demonstrate fundamental HTML skills, including structuring text with headings and paragraphs, working with lists (ordered and unordered), displaying images, and establishing internal navigation using relative file paths. 

The site currently focuses purely on structural layout and content, with styling to be added in future modules.

## Skills Demonstrated
* **HTML Boilerplate & Structure:** Utilizing correct document tags (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
* **Content Formatting:** Implementing headings (`<h1>`, `<h2>`, `<h3>`) and paragraph (`<p>`) tags appropriately.
* **Lists:** Constructing organized data using unordered lists (`<ul>`) for ingredients and ordered lists (`<ol>`) for step-by-step instructions.
* **Hyperlinks & File Paths:** Managing navigation between the main index page and individual recipe pages using relative directory paths (`recipes/filename.html` and `../index.html`).
* **Version Control:** Managing project progression locally and remotely via Git and GitHub.
